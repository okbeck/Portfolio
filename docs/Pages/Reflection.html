Comparing Website Development Methods

Challenges for New Learners

When I first began web development, memorizing all of the tags and Git commands was challenging to me. Then when you go further into development or go to professional pages, you need to be aware of WCAG guidelines. For wordpress, learning the website and tools was challenging. I also found it difficult to actually customize anything, but that is where creating your own theme comes in.

Advantages of Each Method

GitHub pages is more customiziable, and there are more tools to use to assist in development. You can have integration tests built in to ensure new features do not break the page. Wordpress can create professional websites fairly quickly. It can also be cheaper or easier to maintain.

Challenges When Modifying Layout and Organization

GitHub pages has the dreaded misalignment in HTML and CSS, which can be very difficult to debug.

Skills Required for Each Approach

GitHub pages requires knowledge of programming languages, file organization, and repository management. Wordpress requires knowledge of layout design, navigation structure, and content organization.

Future Skills to Improve This Website

To improve the website in the future, I would like to add a backend so there can be more interaction. For front-end, i would want a more appealing design and layout.
